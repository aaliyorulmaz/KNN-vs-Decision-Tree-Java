# KNN vs Decision Tree - Java Implementation

## 📌 Overview
This project implements two fundamental machine learning classification algorithms:

- K-Nearest Neighbors (KNN)
- Decision Tree

The goal is to predict customer product category preferences based on demographic and transaction data.

Unlike typical ML projects, this implementation is built **from scratch in Java** without using any external machine learning libraries.

---

## 🎯 Objectives
- Understand supervised learning and classification
- Implement KNN and Decision Tree algorithms manually
- Apply Object-Oriented Programming (OOP) principles:
  - Encapsulation
  - Inheritance
  - Polymorphism
  - Interfaces
- Compare algorithm performance using:
  - Accuracy
  - Execution Time
  - Error Analysis

---

## 📊 Dataset
- Real-world anonymized retail dataset (Kocaeli - March 2017)
- 5093 transaction records
- Features:
  - Gender
  - Total Price
  - Brand
- Target:
  - Product Category
