# KNN vs Decision Tree - Java Implementation

## 📌 Overview
This project implements two machine learning classification algorithms from scratch:

- K-Nearest Neighbors (KNN)
- Decision Tree

The goal is to predict customer product category preferences based on demographic and transaction data.

This project is built using **pure Java** without any external machine learning libraries, focusing on understanding core algorithm logic and Object-Oriented Programming principles.

---

## 🎯 Objectives
- Implement supervised learning algorithms from scratch
- Apply OOP principles:
  - Encapsulation
  - Inheritance
  - Polymorphism
  - Interfaces
- Compare algorithm performance

---

## 📊 Dataset
- Real-world retail dataset (Kocaeli - 2017)
- 5093 records
- Features:
  - Gender
  - Total Price
  - Brand
- Target:
  - Product Category
